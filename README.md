# AI 학습 노트

## 목차

[1단계: AI의 큰 그림과 작동 원리 파악]

1. AI, 머신러닝, 딥러닝, 강화학습의 개념과 차이점 (개념 잡기)

2. AI 모델의 종류와 기본 작동 원리 (지도학습, 비지도학습, 선형 회귀, 분류 등)

[2단계: 프레임워크 실습과 블랙박스 해소]

3. PyTorch 기초 입문 (텐서 연산, 자동 미분(Autograd), 모델 구성 방법)

4. PyTorch를 이용해 간단한 딥러닝 모델 구축해보기 (OpenCV 경험을 살려 CNN으로 이미지 분류기부터 만들어보시는 것을 강력히 추천합니다.)

5. 머신러닝/딥러닝 핵심 원리 직접 구현해 보기 (C++이나 Python의 순수 행렬 연산만으로 아주 간단한 신경망의 '순전파/역전파'를 짜보면 알고리즘적 호기심이 완벽히 채워집니다.)

[3단계: 모델의 확장과 실무 적용]

6. 기존 모델에 나만의 지식 확장시키기 1: 파인튜닝 (Fine-Tuning) (LoRA 등 가벼운 파인튜닝 기법으로 텍스트/이미지 모델의 가중치 업데이트해 보기)

7. 기존 모델에 나만의 지식 확장시키기 2: RAG (검색 증강 생성) (파인튜닝 없이 외부 데이터를 AI에게 참조시키는 방법 학습)

[4단계: 최신 AI 엔지니어링 및 애플리케이션 개발]

8. 에이전트(Agent)란 무엇인가? (LLM이 스스로 계획하고 도구를 사용하는 원리)

9. MCP(Model Context Protocol)란 무엇인가? (다양한 데이터 소스와 도구를 모델에 표준화된 방식으로 연결하는 구조 이해)

10. MCP, 에이전트, RAG를 융합하여 나만의 AI 비서 만들어보기 (최종 프로젝트)
